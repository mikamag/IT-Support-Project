# IT Support Professional Certificate Capstone Project

## Project Overview

This repository contains all planning documents, research, deliverables, and supporting materials created during the completion of the IT Support Professional Certificate Capstone Project.

The project focuses on designing and documenting a complete IT infrastructure solution for **Ubuntu Innovations (Pty) Ltd**, a technology company consisting of 25 employees. The objective is to apply core IT Support concepts including business requirements analysis, hardware and software planning, networking, troubleshooting, security, documentation, and project collaboration.

---

## Project Objectives

The primary objectives of this project are:

* Analyze business and technical requirements
* Design a scalable IT infrastructure solution
* Recommend appropriate hardware and software resources
* Develop a network topology for the organization
* Create enterprise troubleshooting procedures
* Implement security best practices
* Document all findings professionally
* Collaborate effectively as a project team using GitHub

## Company Scenario

Ubuntu Innovations (Pty) Ltd is a growing technology company with 25 employees working across multiple departments.

The company requires:

* Reliable internet connectivity
* Secure wired and wireless networking
* Centralized file storage
* Shared printing services
* Business-grade hardware
* Standardized software deployment
* Data backup and recovery procedures
* IT support and troubleshooting documentation

The proposed solution has been designed to support current operational requirements while remaining scalable for future growth.
